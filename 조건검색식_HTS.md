# HTS 조건검색식 — 종가 베팅 종목 스크리닝

## 대상 HTS
- 키움증권 영웅문 (조건검색)
- 이베스트 xingAPI / KB증권 M-able
- (한국투자증권 eFriend Plus의 명칭은 다를 수 있음)

---

## 1. 메인 조건검색식 (영웅문 기준)

### 1-1. 가격 조건
```
[A] 시세분석 > 가격조건 > 신고가
    ▸ 기간: 60일 신고가
    ▸ 갱신: 당일

[B] 시세분석 > 가격조건 > 등락률
    ▸ 일간 등락률 5% 이상 ~ 25% 이하
    ▸ (과도한 급등은 추격 매수 위험)
```

### 1-2. 거래량/거래대금 조건
```
[C] 시세분석 > 거래량 > 거래대금
    ▸ 당일 거래대금 500억원 이상
    ▸ (코스닥은 200억원 이상으로 조정 가능)

[D] 시세분석 > 거래량 > 거래량비율
    ▸ (당일 거래량) / (60일 평균 거래량) ≥ 3
```

### 1-3. 종가 강도 조건 (장 막판 매수세)
```
[E] 시세분석 > 가격조건 > 종가/고가 비율
    ▸ 종가 ≥ 당일 고가 × 0.97
    ▸ (장중 고가의 3% 이내에서 마감)

[F] 시세분석 > 캔들패턴 > 장대양봉
    ▸ 캔들 몸통 길이 ≥ 일 등락폭의 70%
```

### 1-4. 이격도 조건 (추격 매수 방지)
```
[G] 기술적분석 > 이격도 > 5일 이격도
    ▸ 100 ≤ 5일 이격도 ≤ 115
    ▸ (5일선 대비 +15% 이내)
```

### 1-5. 시장 환경 필터 (옵션)
```
[H] 외부지표 > 코스피 지수
    ▸ 코스피 종가 ≥ 5일 이평선

[I] 외부지표 > 코스닥 지수
    ▸ 코스닥 종가 ≥ 5일 이평선
```

### 최종 결합
```
A AND B AND C AND D AND E AND F AND G AND H AND I
```

## 2. 보조 조건검색식 — 5일선 눌림목 회복

```
[A] 5일 이동평균선과 종가의 비율
    ▸ 종가 ≥ 5일 이평 × 1.00
    ▸ 종가 ≤ 5일 이평 × 1.03

[B] 직전 1~3일 봉 패턴
    ▸ 최근 3일 중 1일 이상 5일선 이탈 후 회복

[C] 거래대금 200억 이상 + 거래량 ≥ 60일 평균 ×2
```

---

## 3. 보조 조건검색식 — 박스권 돌파

```
[A] 박스권 정의 (최근 5~10일)
    ▸ 최근 10일 고가 - 최근 10일 저가 ≤ 평균가 × 0.08
    ▸ (변동성 8% 이내 횡보)

[B] 돌파 조건
    ▸ 종가 ≥ 최근 10일 고가
    ▸ 당일 등락률 ≥ 5%

[C] 거래량 폭증
    ▸ 당일 거래량 ≥ 최근 10일 평균 × 3
```

## 4. 실시간 알림 설정 (오후 2:30 이후)

### 영웅문 실시간 검색
```
조건검색 > 실시간 등록
실행 주기: 1분
시간: 14:30 ~ 15:30 한정
알림: 팝업 + 사운드
```

### Push 알림 (모바일)
```
M-able / 영웅문 모바일
종목 등록 > 가격 알림
- 신고가 도달
- 거래대금 임계치 돌파
- 5분봉 종가 박스 상단 돌파
```

## 6. 주의 사항

1. **조건검색만 믿지 말 것** — 차트와 호가, 테마 흐름 직접 확인 필수
2. **변동성 큰 종목 주의** — 정치 테마, 작전주는 제외 권장
3. **시총 필터 추가 권장** — 최소 1,000억 이상 (유동성 확보)
4. **관리종목·투자유의·거래정지 이력** 종목 제외
5. **공매도 잔고 비율** 5% 이상 종목 주의 (역방향 압력)
