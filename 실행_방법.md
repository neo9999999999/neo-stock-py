# 실행 방법

## 1. 가상환경 활성화 (매번 필요)

```bash
cd "/Users/neo/Desktop/종가베팅_분석/03_구현"
source ../.venv/bin/activate
```

---

## 2. 웹 대시보드 실행

```bash
streamlit run src/app.py
```

→ 브라우저에서 자동으로 http://localhost:8501 열림.

### 페이지

| 페이지 | 용도 |
|---|---|
| **오늘의 추천** | 임의 일자 기준 5대 조건 충족 종목 스크리닝 |
| **백테스트 결과** | 기본 파라미터의 5년치 백테스트 결과 + 차트 |
| **워크포워드** | OOS(out-of-sample) 검증 결과 + 윈도우별 최적 파라미터 |
| **파라미터 시뮬레이터** | 임계치를 직접 조정하며 즉시 백테스트 |
| **사례 검증** | 보고서 사례 4건 차트로 확인 |

---

## 3. 데이터 재다운로드 (보통 불필요)

```bash
python src/download_data.py --start 20210101 --end 20260517 --top_n 300
```

캐시된 데이터가 있으면 스킵. 강제 재다운로드는 `data/ohlcv/` 폴더 삭제 후 실행.

---

## 4. 백테스트 + 최적화 재실행

```bash
python src/run_full.py --start 20210101 --end 20260517 --out results
```

옵션:
- `--skip_grid`: 그리드 서치 생략
- `--skip_wf`: 워크포워드 생략

---

## 5. 보고서 재생성

```bash
python src/generate_report.py
```

→ `04_검증_분석/05_백테스트_결과보고서.md` 생성.

---

## 6. 디렉토리 구조

```
03_구현/
├── src/
│   ├── data_loader.py      # FDR 기반 데이터 로더 + 캐시
│   ├── engine.py           # 시그널 + 백테스트 엔진
│   ├── walk_forward.py     # 그리드 서치 + 워크포워드
│   ├── download_data.py    # 5년치 다운로드 (한 번만)
│   ├── run_full.py         # 풀 파이프라인 실행
│   ├── generate_report.py  # 마크다운 보고서 생성
│   └── app.py              # Streamlit 웹 대시보드
├── data/
│   ├── ohlcv/              # 종목별 일별 OHLCV (parquet)
│   ├── listing/            # KRX 상장 리스팅
│   └── index/              # KOSPI/KOSDAQ 지수
└── results/
    ├── base_metrics.json   # 기본 파라미터 결과
    ├── base_trades.parquet # 거래 내역
    ├── grid_search.parquet # 그리드 서치 전체 결과
    ├── wf_summary.json     # 워크포워드 요약
    └── wf_oos_trades.parquet # OOS 거래 내역
```
